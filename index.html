Mandelbrot

Description

This page is intended to explore the Mandelbrot set. Below is a canvas with the Mandelbrot set already rendered. A number field allows you to set the zoom of that picture. Clicking on the Mandelbrot set image will paint a red square indicating where the center of the zoom will be; Default is the center (0,0). There are several settings enabled through a list of checkboxes. Several button below will allow resizing the image. It is recommended that you only use the larger setting when you find a space you want to see in greater detail. Use the default size as a view finder of sorts.

Instructions

1. Click on the Mandelbrot canvas. This will set the center of your next image so plan accordingly.
2. Set the zoom.
3. (Optional) Check the box that says Random Layers. This will only take affect after the draw button is clicked.
4. Click draw. (On larger sizes this might take up to 30 seconds.)
5. (Optional) If the random colors are not to your liking click Reset Random Layers.
6. Repeat steps starting at 1 until an image you want to save is found.
7. (Optional) Click the check box labeled Double Resolution.
8. Click on a size. (This will take some time as it is about to redraw the image)
9. Right click on image for save options.
